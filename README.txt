Использование

В терминале в директории решения запустите

dotnet run <файл заказов> <район> <время первой доставки> <файл логов> <файл результата>

Например,

dotnet run заказы.txt Central "2001-12-12 12:00:00" log.txt result.txt

Формат строк файла заказов:
<номер заказа>,<вес заказа>,<район>,<дата и время доставки>
